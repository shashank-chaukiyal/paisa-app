# Paisa App 💰

Paisa Tracker Application to ease your process and spending

## Overview

Paisa is a personal finance management application designed to help you track your spending, manage your budget, and gain insights into your financial habits. Whether you're looking to control your expenses or understand where your money goes, Paisa provides an intuitive interface to manage your personal finances effectively.

## Features

- 📊 **Expense Tracking** - Record and categorize your daily expenses
- 💳 **Budget Management** - Set budgets and monitor spending against them
- 📈 **Financial Analytics** - Visualize your spending patterns with charts and reports
- 🏷️ **Category Organization** - Organize expenses into customizable categories
- 🔍 **Search & Filter** - Easily find and filter your transactions
- 💾 **Data Persistence** - Secure storage of your financial data

## Getting Started

### Prerequisites

- Python 3.8 or higher
- pip (Python package manager)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/shashank-chaukiyal/paisa-app.git
cd paisa-app
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python -m Paisa_app.paisa
```
